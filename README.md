# How to run

You will need valid API key from [TMDB API](https://developer.themoviedb.org/reference/intro/getting-started)

```
git clone
cd MovieInfoSearchWeb
(
echo const API_KEY_TMDB = "${YOUR_API_KEY}";
echo(
echo export default API_KEY_TMDB;
) > keys.js
index.html
```
